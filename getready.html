.NET Core

Початок роботи

.NET Core - вільна, кроссплатформенна реалізація .NET Framework із відкритим кодом, створена компанією Microsoft. Платформа створена для розробки сайтів та веб-додатків.

Для роботи необхідні дві речі: встановлення платформи .NET Core і
текстовий редактор для написання коду.

Встановити платформу .NET Core можна, завантаживши її із офіційного сайту.

Для Windows-розробників найкращим середовищем для розробки програм є Microsoft Visual Studio. VS дозволяє розробляти як консольні програми, так і програми з графічним інтерфейсом, в тому числі з підтримкою технології Windows Forms, а також веб-сайти, веб-застосунки, веб-служби як в рідному, так і в керованому кодах для всіх платформ, що підтримуються Microsoft Windows, Windows Mobile, Windows Phone, Windows CE, .NET Framework, .NET Compact Framework, .NET Core та Microsoft Silverlight.

Visual Studio Code - безкоштовний текстовий редактор із відкритим кодом для Windows, Linux та OSX.

Також прекрасним вибором є текстові редактори Atom та Sublime Text. Взагалі, для розробки вам підійде будь-який текстовий редактор.
